[admin 토큰]
통합회원가입(scope로 관리자와 사용자등 분리가능)
- POST /v1/users
{
    client_id: 'gncloud-kr'
    scope: ['user'],
    id: ...
}
회원정보조회
- GET /v1/users/{userId}

회원조회(검색)
- GET /v1/users(?search=검색어)

회원정보수정
- PUT /v1/users/{userId}

회원삭제(통합탈퇴)
- DELETE /v1/users/{userId}




토큰전체조회
- GET /v1/tokens

토큰 1개 조회
- GET /v1/tokens/{tokenId}

토큰삭제(유출시사용)
- DELETE /v1/tokens/{tokenId}

Model User {userId, password, email, name, registerDate, phone, address, company}
Model Scope : {id, description, isDefault}
Model Client {clientId, clientSecret, description, domain}

Model UserClientScope {clientId, userId, scope}
#로그인할때 만들어지는 레코드
Model Token {id, clientId, userId, createTime, expireDate}
